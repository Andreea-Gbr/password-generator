# 🔐 Password Generator Utility

✨ A clean and efficient Python script that generates secure, randomized passwords based on user-defined length.

## 🌟 Features
- 🧠 **Smart Validation:** Prevents errors by checking for negative numbers or zero.
- 🎲 **High Entropy:** Combines uppercase, lowercase, digits, and special characters for maximum security.
- 💪 **Robust:** Built-in error handling for non-numeric inputs so the script won't crash.

## ⚙️ Prerequisites
- 🐍 Python 3.6 or higher

## 🚀 Installation & Usage

1. **⬇️ Clone or download the script:**
   Save `password_gen.py` to your machine.

2. **▶️ Run the script:**
   ```bash
   python password_gen.py
